🎯 사용 시나리오

고객용 퀴즈 시나리오

1. 온라인 마케팅

광고 → 랜딩페이지 → 퀴즈 → 전환

2. 소셜미디어

인스타그램/페이스북 → 퀴즈 → 리드 생성

3. 블로그/콘텐츠

골프 관련 콘텐츠 → 퀴즈 → 참여 유도

4. 이메일 마케팅

뉴스레터 → 퀴즈 → 재참여 유도

오피스용 퀴즈 시나리오

1. 전화 상담

고객 전화 → 상담 → 퀴즈 → 제안

2. 매장 방문

매장 방문 → 클럽 체크 → 퀴즈 → 시타

3. 시타 상담

시타 전 → 상세 퀴즈 → 맞춤 제품 추천

4. 구매 결정

시타 후 → 최종 퀴즈 → 구매 결정
