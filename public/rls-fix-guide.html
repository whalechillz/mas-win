Supabase RLS 정책 수정 방법

1. Supabase 대시보드 접속

https://app.supabase.com 로 이동하여 로그인

2. RLS 정책 확인

  1. 좌측 메뉴에서 "Authentication" → "Policies" 클릭
  2. bookings 테이블과 contacts 테이블 확인
  3. RLS가 활성화되어 있다면 정책 추가 필요

3. RLS 정책 추가 (bookings 테이블)

-- INSERT 정책 (누구나 예약 가능)
CREATE POLICY "Enable insert for all users" ON bookings
FOR INSERT WITH CHECK (true);

-- SELECT 정책 (선택사항)
CREATE POLICY "Enable read for all users" ON bookings
FOR SELECT USING (true);

4. RLS 정책 추가 (contacts 테이블)

-- INSERT 정책 (누구나 문의 가능)
CREATE POLICY "Enable insert for all users" ON contacts
FOR INSERT WITH CHECK (true);

5. 또는 RLS 비활성화

각 테이블에서 RLS 토글을 OFF로 설정 (보안상 권장하지 않음)
