# Detail 이미지 콘텐츠 입력 가이드

## 📋 개요

Detail 이미지 섹션은 제품 페이지에서 **8컷 상세 이미지**로 표시되며, 각 이미지마다 **제목**과 **설명**을 입력해야 합니다.

---

## ✅ 입력이 필요한 이유

1. **사용자 경험 향상**: 이미지만으로는 부족한 정보를 텍스트로 보완
2. **SEO 최적화**: 검색 엔진이 제품 특징을 이해할 수 있도록 도움
3. **전환율 향상**: 명확한 설명으로 구매 결정을 돕습니다

---

## 📝 입력 방법

### 1단계: 관리 페이지 접속
1. `/admin/products` 접속
2. `secret-weapon-black-muziik` 제품 선택
3. **수정** 버튼 클릭

### 2단계: Detail 탭 선택
1. **"제품 이미지 관리"** 섹션에서
2. **"Detail"** 탭 클릭

### 3단계: 이미지 추가
1. **"📷 이미지 업로드"** 또는 **"🖼️ 갤러리에서 선택"** 버튼 사용
2. Detail 이미지 8개 추가 (권장)

### 4단계: 콘텐츠 입력
1. 이미지 추가 후, **"Detail 이미지 텍스트 편집"** 섹션이 자동으로 나타납니다
2. 각 이미지마다:
   - **제목**: 기술 특징의 핵심 (예: "임팩트시 역토크 방지")
   - **설명**: 상세한 설명 (예: "헤드의 직진성과 방향성을 향상")

## 🔧 조정 가능한 항목

### 1. 이미지 순서 변경
- Detail 탭에서 **↑** 또는 **↓** 버튼으로 순서 변경
- 순서는 제품 페이지에 표시되는 순서와 동일합니다

### 2. 이미지 교체
- **"갤러리에서 선택"** 버튼으로 다른 이미지로 교체 가능
- 기존 이미지는 Storage에 유지됩니다

### 3. 텍스트 수정
- 언제든지 **제목**과 **설명**을 수정할 수 있습니다
- 수정 후 **저장** 버튼 클릭

### 4. 이미지 제거
- **"제외"** 버튼으로 목록에서 제거
- Storage 파일은 유지되므로 나중에 다시 추가 가능
