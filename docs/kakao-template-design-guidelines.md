## 1. 이모지/특수문자 사용 규칙

### ✅ 안전한 이모지 (검수 통과 확인)
- **☎** (전화 이모지): 연락처 섹션에만 1회 사용 권장
  - 검수 통과 확인됨
  - 여러 번 사용 시 문제 발생 가능

### ❌ 문제가 될 수 있는 이모지
- **📞** (다른 전화 이모지): 검수에서 문제 발생 가능
- **📱**, **📲** 등 다른 통신 관련 이모지: 일관성 문제

### ✅ 권장 기호 사용

#### 정보 표시
- **•** (불릿 포인트): 가장 안전하고 일관성 있음
- **▶** (삼각형): 사용 가능하지만 • 권장
- **→** (화살표): 사용 가능

#### 섹션 구분
- **[ ]** (대괄호): 섹션 제목에 사용
  - 예: `[제품 정보]`, `[예약 정보]`, `[안내사항]`, `[연락처]`

#### 안내사항
- **•** (불릿): 안내사항 리스트에 사용 (⊙ 대신)
- **⊙** (원 안 점): 사용 가능하지만 • 권장

```

[본문 내용]

[섹션 제목]

수원시 영통구 법조로149번길 200
TEL 031-215-0013
OPEN 09:00~18:00(월~금)
```

### 섹션 구성 원칙
1. **인사말**: 항상 첫 줄
2. **본문**: 주요 내용
3. **정보 섹션**: `[제품 정보]`, `[예약 정보]` 등
4. **안내사항**: `[안내사항]` 섹션
5. **연락처**: 마지막에 `[연락처]` 섹션으로 구분

## 3. 기호 사용 일관성

### 변경 전 → 변경 후

| 기존 | 권장 | 이유 |
|------|------|------|
| ▶ | • | 더 안전하고 일관성 있음 |
| ⊙ | • | 검수 안전성 향상 |
| ☎ (여러 번) | ☎ (1회만) | 연락처 섹션에만 사용 |
| 📞 | ☎ | 검수 통과 확인된 기호 사용 |

---

## 4. 검수 통과 팁

### ✅ 권장 사항
1. **이모지 최소화**: ☎는 연락처에만 1회 사용
2. **일관된 기호**: 모든 템플릿에서 • 사용
3. **명확한 섹션 구분**: [ ]로 섹션 제목 표시
4. **간결한 문구**: 불필요한 장식 제거

### ❌ 피해야 할 사항
1. **이모지 남용**: 여러 종류의 이모지 혼용
2. **기호 불일치**: 템플릿마다 다른 기호 사용
3. **복잡한 구조**: 너무 많은 섹션과 기호

## 5. 회사 정보 배치 가이드라인

### 배치 옵션 비교

#### 옵션 1: 본문 하단에 배치 (기존 방식)
**장점:**
- 가독성 좋음 (메시지 버블 안에 크게 표시)
- 기존 템플릿과 일관성 유지
- 변수 사용 가능 (필요시)

**단점:**
- 본문이 길어짐
- 모든 템플릿에 반복 입력 필요

#### 옵션 2: 부가정보(선택사항)에 배치 (권장)
**장점:**
- 본문이 간결해짐
- 모든 템플릿에서 일관성 있게 표시
- 변수 불필요 (고정 정보이므로 적합)
- 템플릿 관리 용이

**단점:**
- 메시지 하단에 작게 표시됨
- 가독성이 약간 떨어질 수 있음

### 최종 권장: 부가정보(선택사항)에 배치 ✅

**이유:**
1. **일관성**: 모든 템플릿에서 동일한 위치에 표시
2. **간결성**: 본문이 더 간결하고 핵심 내용에 집중
3. **관리 용이성**: 한 곳에서 관리, 수정 시 모든 템플릿에 자동 반영
4. **변수 불필요**: 고정 정보이므로 부가정보에 적합
