# 고객 이미지 누락 문제 일괄 수정 계획

## 문제 상황

- 이남구 등 50명 이상의 고객에서 고객 이미지 관리 - 미디어에 이미지가 표시되지 않음
- 갤러리에서 이미지 선택 시에는 이미지가 표시됨
- 원인: `file_path`에 파일명이 없거나 `ai_tags`에 `customer-{id}` 태그가 없는 경우

## 해결 방안

### 1. API 개선 (완료)
- `verifyFileExists` 함수로 인한 이미지 제외 문제 해결
- 파일 존재 확인을 건너뛰고 모든 메타데이터 이미지 포함
- `file_path`에 파일명이 없는 경우 `filename`으로 자동 수정

### 2. 일괄 수정 스크립트 (선택사항)
- 모든 고객의 이미지 메타데이터 일괄 수정
- `file_path`에 파일명 추가
- `ai_tags`에 `customer-{id}` 태그 추가
- `cdn_url` 업데이트

## 스크립트 실행 방법

```bash
# 일괄 수정 스크립트 실행 (선택사항)
node scripts/fix-all-customers-missing-images.js
```

## 예상 효과

- 모든 고객의 이미지가 고객 이미지 관리 화면에 표시됨
- 갤러리와 고객 이미지 관리 화면에서 동일한 이미지 표시
- 데이터 일관성 향상
