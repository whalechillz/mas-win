## 주간 콘텐츠 개수 계산

### 주당 허브 콘텐츠
- **월요일**: 1개 (비거리 비법 시리즈)
- **화요일**: 1개 (유용한 정보)
- **수요일**: 1개 (비거리 비법 시리즈)
- **목요일**: 1개 (제품/브랜드)
- **금요일**: 1개 (비거리 비법 시리즈)
- **총계**: 주당 5개

### 월간 허브 콘텐츠
- 주당 5개 × 4주 = **월 20개**
- 비거리 비법 시리즈: 월 12개 (월/수/금 × 4주)
- 유용한 정보: 월 4개 (화요일 × 4주)
- 제품/브랜드: 월 4개 (목요일 × 4주)

### 연간 허브 콘텐츠
- 월 20개 × 12개월 = **연간 약 240개** (휴일 제외 시 약 260개)

## 콘텐츠 상세 계획

### 1. 킬러 콘텐츠: 드라이버 비거리 늘리기 비법 시리즈

#### 시리즈 구조
- **하나의 허브 콘텐츠**로 시리즈 연재 (관리 편의성)
- 각 편은 독립적인 블로그 포스트로 발행
- 허브에서 시리즈 전체 관리

#### 주제 구성 (golfdistillery.com, golfclubsadvisor.com 기반)

**Phase 1: 기초편 (1-4주)**
1. 스윙 속도 향상의 5가지 비법
2. 임팩트 포인트 최적화로 비거리 20m 늘리기
3. 시니어 골퍼를 위한 비거리 향상 특별 가이드
4. 드라이버 헤드 스피드 측정과 개선 방법

**Phase 2: 중급편 (5-8주)**
5. 스윙 궤적과 비거리의 관계
6. 백스윙에서 비거리를 만드는 비밀
7. 다운스윙 가속화 기술
8. 임팩트 순간의 파워 전달법

**Phase 3: 고급편 (9-12주)**
9. 클럽 피팅과 비거리 최적화
10. 샤프트 선택이 비거리에 미치는 영향
11. 헤드 로프트와 비거리의 관계
12. 시니어 골퍼의 비거리 향상 사례 연구

#### 콘텐츠 작성 가이드
- **타겟**: 피팅 선호 시니어, 테크 선호 얼리어답터 (페르소나를 직접 언급하지 않지만 그들을 목표로)
- **톤**: 전문적이지만 이해하기 쉬운 설명
- **구조**: 문제 제시 → 원인 분석 → 해결 방법 → 실전 팁
- **SEO**: 네이버 검색 최적화 (롱테일 키워드 활용)
- **소스**: golfdistillery.com, golfclubsadvisor.com 번역 및 한국형 최적화
- **후킹**: 가장 후킹되는 제목과 이미지 제작

- 타겟 페르소나에 맞는 주제 선택

### 2. 콘텐츠 작성
- 원문 번역 및 한국형 최적화
- 타겟 페르소나에 맞는 톤 조정
- SEO 키워드 최적화
- 후킹 제목 및 이미지 제작

### 3. 허브 콘텐츠 생성
- 허브 콘텐츠 기본 정보 입력
- 요약 및 본문 작성
- 채널별 베리에이션 준비

### 4. 채널별 베리에이션 생성
- 블로그: 상세 가이드 작성
- SMS: 핵심 정보 요약

- 카카오톡: 인포그래픽 제작

### 5. 발행 및 관리
- 발행 일정 확인

- 성과 추적 및 분석

- 주당 허브 콘텐츠: 5개

- SMS: 주당 5개
- 네이버 블로그: 주당 5개
