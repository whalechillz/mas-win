# 📱 직원용 네이버 블로그 관리 가이드

## 🚀 빠른 시작 (3단계)

### 1️⃣ 로그인
1. https://win.masgolf.co.kr/admin 접속
2. 아이디: `admin` / 비밀번호: `1234`

### 2️⃣ 블로그 관리 접속
1. 좌측 메뉴 "마케팅 콘텐츠" 클릭
2. "✨ 블로그 관리 (간편)" 탭 선택 (기본 선택됨)

### 3️⃣ 글 작성 시작
1. 우측 상단 "+ 새 주제 추가" 클릭
2. 주제 입력 (예: "시니어 골퍼를 위한 MASGOLF 드라이버")
3. 담당자 선택
4. "추가" 클릭 → 3개 계정용 글이 자동 생성됨

## 📋 실제 업무 프로세스

### 월요일: 주간 계획
```
1. 팀 회의에서 이번 주 주제 5-7개 선정
2. 각 주제마다 담당자 배정
3. 시스템에 주제 추가
```

### 화~수요일: 글 작성
```
1. 담당 글의 상태를 "아이디어" → "작성중" 변경
2. 3가지 다른 관점으로 글 작성:
   - [실제 후기] - 고객 경험 중심
   - [전문가 팁] - 교육/정보 중심  
   - [비교 분석] - 경쟁사 비교 중심
```

### 목~금요일: 발행
```
1. 상태를 "작성중" → "발행준비" 변경

2. 발행 방법 선택:
   A. 🕐 예약 발행 (권장)
      - "예약 발행" 버튼 클릭
      - 네이버 블로그에서 예약 설정
      - 상태가 "예약완료"로 변경됨
      - 발행 후 "✅ 발행 완료 처리" 클릭
   
   B. ⏰ 즉시 발행
      - URL 입력창에 바로 입력
      - Enter 키 누르면 "발행완료"로 변경

3. 발행 시간:
   - 09:00 - mas9golf 계정
   - 14:00 - massgoogolf 계정
   - 19:00 - massgoogolfkorea 계정

4. 조회수 추적
```

## 🕐 네이버 예약 발행 가이드

### 예약 발행의 장점
- 📝 여러 글을 미리 작성해두고 예약
- ⏰ 정확한 시간에 자동 발행
- 🏖️ 휴가에도 발행 가능
- 📈 일정대로 꾸준히 발행

### 예약 설정 방법
1. 네이버 블로그 접속
2. 글쓰기 → 내용 작성
3. "발행" 버튼 옆 "▼" 클릭
4. "예약 발행" 선택
5. 날짜와 시간 설정
6. "예약" 버튼 클릭

### 시스템 관리
1. 예약 완료 시 "예약완료" 상태로 변경
2. 발행 후 "✅ 발행 완료 처리" 클릭
3. URL 입력 후 "발행완료"로 변경

### 담당자별 계정 배정
- **제이**: mas9golf (주로 오전 발행)
- **스테피**: massgoogolf (주로 오후 발행)
- **나과장**: 유동적 배정
- **허상원**: massgoogolfkorea (주로 저녁 발행)

## 🎯 키워드 전략

- 마스골프, MASGOLF, 마스골프 드라이버
- 난이도: 낮음 / 우선순위: 높음

- 난이도: 중간 / 우선순위: 높음

- 고반발 드라이버, 비거리 드라이버
- 난이도: 높음 / 우선순위: 낮음

## 📞 문제 해결

### 자주 묻는 질문

**Q: 편집 버튼이 안 눌려요**
A: 페이지 새로고침 (F5) 후 다시 시도

**Q: 상태 변경이 안 돼요**  
A: 드롭다운 클릭 후 2초 대기

A: Enter 키를 꼭 눌러주세요

**Q: 조회수가 안 올라요**
A: 제목, 태그, 발행 시간 점검
