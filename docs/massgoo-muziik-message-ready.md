## 📋 메시지 생성 방법

### 방법 1: Playwright 스크립트 자동 생성

```bash
node scripts/create-massgoo-muziik-survey-messages.js
```

### 방법 2: 수동 생성

1. **SMS/MMS 에디터 접속**
   - URL: `/admin/sms`
   - 새 메시지 생성

2. **메시지 타입**: MMS 선택

3. **메시지 내용 입력** (위의 내용 복사)

4. **이미지 선택**
   - "갤러리에서 선택" 클릭
   - `composed-1-black-beryl-1764776071615.png` 검색 및 선택

5. **수신자 추가**
   - "스탭진 추가" 버튼 클릭

6. **예약 시간 설정**
   - "예약 사용" 체크
   - 날짜: 오늘
   - 시간: 15:00, 15:01, 15:02 (각각 다른 메시지)

7. **저장**
