## 📋 작업 순서

### 오전 7시~9시 (권장 시간)

1. **허브 콘텐츠 자동 생성**
   ```bash
   node scripts/auto-create-hub-content.js
   ```
   - 허브 콘텐츠는 자동으로 생성됩니다
   - 생성 후 작성일지에 기록

2. **프로필 이미지 준비**
   - 갤러리에서 이미지 선택 또는 AI로 생성
   - 대표폰: 따뜻한 톤 (골드·브라운)
   - 업무폰: 쿨 톤 (블루·그레이)

3. **프로필 업데이트 (수동)**
   - 카카오톡 프로필 관리 페이지 접속
   - 대표폰: 배경 → 프로필 순으로 업데이트
   - 업무폰: 배경 → 프로필 순으로 업데이트
   - 각각 메시지 입력

4. **카카오 피드 생성 (수동)**
   - 카카오 채널 관리 페이지 접속
   - 계정 1: 이미지 업로드 + 캡션 입력
   - 계정 2: 이미지 업로드 + 캡션 입력
   - 발행

5. **반영 확인**
   - 업로드 후 5분 내 피드 반영 여부 확인
   - 작성일지 업데이트
