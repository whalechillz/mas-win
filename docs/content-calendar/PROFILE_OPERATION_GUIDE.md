# MASGOLF 카카오 프로필 운영 가이드

## 📋 개요

MASGOLF의 두 개 카카오 계정을 각각 **페르소나별 브랜드 확장 채널**로 운영합니다.

- **010-6669-9000**: 시니어 중심 감성형 브랜딩 (대표 개인폰)
- **010-5704-0013**: 하이테크 중심 혁신형 브랜딩 (업무폰)

## 🎯 운영 목적

1. 피드 노출 빈도를 높여 브랜드 인지 강화
2. 시니어 타깃의 '공감 신뢰' + 중장년층의 '기술 신뢰' 동시 확보
3. 매일 4개의 프로필 콘텐츠(계정당 배경 1 + 프로필 1)로 브랜드 스토리 지속 노출

## 📊 운영 기본 구조

| 항목 | 대표 개인폰 (시니어 중심) | 마스골프 업무폰 (테크 중심) |
|------|------------------------|------------------------|
| **페르소나** | 감성적 시니어 골퍼 | 하이테크 얼리어답터 |
| **업로드 주기** | 매일 1회 | 매일 1회 |
| **이미지 구성** | 배경 1 + 프로필 1 | 배경 1 + 프로필 1 |
| **콘텐츠 톤** | 따뜻·품격·철학형 | 세련·데이터·혁신형 |
| **문구 방향** | 명언, 감성, 건강 조언 | 팁, 기술, 브랜드 메시지 |

## 📅 운영 방법

### 업데이트 시간
- **매일 오전 7~9시 사이 업데이트** (피드 노출 최적화)
- 프로필사진 → 배경 순으로 변경 시 노출 효과 상승

### 이미지 관리
- **저장 경로**: `originals/daily-branding/kakao/YYYY-MM-DD/account1|account2/background|profile|feed/`
- AI 이미지 생성 시 날짜별 폴더로 정리
- 계정별 색조 일관성 유지:
  - **대표폰**: 따뜻한 톤(골드·브라운)
  - **업무폰**: 쿨 톤(블루·그레이)

### BasePrompt 관리
- **BasePrompt**: 요일별 로테이션되는 한국어 설명 템플릿
- **역할**: 상세 프롬프트 생성의 기반이 되는 템플릿
- **관리 방법**:
  1. **요일별 자동 생성**: ProfileManager에서 "🔄 요일별 자동 생성" 버튼으로 요일별 템플릿 자동 선택
  2. **수동 편집**: "✏️ 편집" 버튼으로 basePrompt 직접 수정
  3. **저장**: "💾 저장" 버튼으로 Supabase에 저장
- **요일별 로테이션**: 매주 같은 요일마다 유사한 테마의 basePrompt 사용
- **주차별 테마 반영**: 주차별 테마에 맞게 basePrompt 조정
- **자세한 내용**: `BASE_PROMPT_MANAGEMENT.md` 참고

### 문구 구성
- **브랜드 표기 (고정)**: 브랜드 인지도를 위해 고정 유지
  - **첫 번째 필드 (7/20)**: "MASSGOO" - 브랜드 표기 (두 계정 모두 동일)
- **상태 메시지 (매일 변경)**: 일일 철학 메시지
  - **두 번째 필드 (13/60)**: 매일 다른 메시지 입력
  - **형식**: (짧은 헤드라인) + (한 문장 철학 메시지)
  - 계정별 톤앤매너 일관성 유지:
    - **대표폰**: 감성적, 따뜻한 메시지 (예: "스윙보다 마음이 먼저다.")
    - **업무폰**: 기술적, 혁신적인 메시지

## 🚀 자동화

### 카카오톡 프로필 업데이트 자동화 ✅

**구현 완료**: Playwright 기반 자동화 스크립트

**프로필 구조** (최종 확정):
- **첫 번째 필드 (7/20)**: "MASSGOO" - 브랜드 표기 (고정)
- **두 번째 필드 (13/60)**: 상태 메시지 - 매일 변경 (예: "스윙보다 마음이 먼저다.")
- **배경 이미지**: 골프장 일몰 등 계절별/요일별 이미지
- **프로필 이미지**: 시니어 골퍼 이미지 (account1) / 젊은 골퍼 이미지 (account2)

**사용 방법**:
1. 카카오 콘텐츠 생성 페이지에서 이미지와 메시지 생성
2. "카카오톡 업로드" 버튼 클릭
3. 브라우저가 자동으로 열리며 프로필 업데이트 진행
4. 수동 확인 후 저장

**환경 변수 설정** (`.env.local`):
```bash

KAKAO_ACCOUNT1_PASSWORD=Zoo100zoo!!
```

**스크립트 직접 실행**:
```bash

```

**자동화 프로세스**:
1. 카카오톡 PC 버전 로그인
2. 프로필 편집 페이지로 이동

## 🎨 프롬프트 관리

### 프롬프트 고정화 원칙
1. **캘린더 JSON에 상세 프롬프트 저장**: 영어로 100자 이상의 상세 프롬프트를 `prompt` 필드에 저장
2. **재생성 시 일관성 유지**: 이미 상세 프롬프트가 있으면 약간만 보강, 없으면 전체 생성
3. **계정별 특화**: 
   - account1: 해변, 산악 명문코스 절경 (제주도, 부산, 설악산, 지리산)
   - account2: 스코틀랜드, 유럽 명문코스 (St. Andrews, Royal Troon, 스페인, 포르투갈)

### 프롬프트 작성 가이드
- **배경 이미지**: 배경/장면이 주인공, 사람은 작게 또는 없음
- **프로필 이미지**: 골퍼가 주인공
- **필수 포함**: 한국 골퍼 명시, 톤 명시, 해상도 명시, 형식 명시
